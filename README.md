# 世德會議助理

一個基於 Apple Silicon 優化的會議語音摘要助手，能夠將會議錄音自動轉錄為逐字稿，並透過本地 LLM 生成結構化摘要。

## 功能特點

- **語音轉文字**：使用 mlx-whisper 進行高效能語音識別，針對 Apple Silicon 優化
- **時間軸逐字稿**：自動產生帶有時間戳記的轉錄文字
- **智慧摘要**：透過 Ollama 本地 LLM 生成結構化會議摘要
- **多種摘要風格**：支援會議摘要、文章摘要、簡短摘要三種模式
- **多格式支援**：可處理 MP3、WAV、M4A、OGG、FLAC 等常見音檔格式
- **隱私優先**：所有處理皆在本機執行，無需上傳至雲端

## 系統需求

- macOS（Apple Silicon M1/M2/M3 系列晶片）
- Python 3.9 或更新版本
- Ollama（本地 LLM 推理引擎）
- 建議至少 16GB RAM

## 快速開始

### 1. 安裝 Ollama

```bash
# 下載並安裝 Ollama
curl -fsSL https://ollama.com/install.sh | sh

# 下載 qwen3:32b-q4_K_M 模型
ollama pull qwen3:32b-q4_K_M
```

### 2. 設定 Python 環境

```bash
# 建立虛擬環境
python3 -m venv venv
source venv/bin/activate

# 安裝依賴套件
pip install -r poc/requirements.txt
```

### 3. 啟動服務

```bash
# 在一個終端機視窗啟動 Ollama 服務
ollama serve

# 在另一個終端機視窗啟動 Web 應用
cd poc
python app.py
```

### 4. 開始使用

在瀏覽器開啟 http://localhost:7860，即可上傳音檔並開始轉錄與摘要。

## 使用說明

1. **上傳音檔**：點擊上傳區域或直接拖曳音檔
2. **選擇摘要風格**：
   - **會議摘要**：產生完整的會議紀錄，包含摘要、重點、待辦事項與決議
   - **文章摘要**：產生約 100 字的摘要與關鍵詞
   - **簡短摘要**：產生 3 句話以內的精簡摘要
3. **等待處理**：依據音檔長度，處理時間約 1-5 分鐘
4. **檢視結果**：轉錄完成後會顯示帶時間軸的逐字稿與 AI 摘要

## 效能參考

| 音檔長度 | 轉錄時間 | 摘要時間 | 總處理時間 |
|---------|---------|---------|-----------|
| 5 分鐘  | ~30 秒  | ~15 秒  | ~45 秒    |
| 30 分鐘 | ~2 分鐘 | ~20 秒  | ~2.5 分鐘 |
| 60 分鐘 | ~4 分鐘 | ~30 秒  | ~4.5 分鐘 |

*測試環境：MacBook Pro M2 Max, 32GB RAM*

## 已知限制

- 目前僅支援 Apple Silicon Mac
- 首次執行需下載約 3GB 的 Whisper 模型
- 長音檔（超過 2 小時）可能需要較長處理時間
- 背景雜訊較多的音檔可能影響轉錄品質
